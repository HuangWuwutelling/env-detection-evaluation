# 环境检测数据评价系统

一个基于 Streamlit 的环境检测数据自动化评价系统，支持样品数据管理、评价标准配置、自动评价和结果导出功能。

## 功能特性

✅ **样品管理**
- 手动录入样品信息
- Excel/CSV 批量导入样品数据
- 样品列表查看和搜索
- 🆕 **专用 Excel 模板下载**（按标准类型自动生成）

✅ **评价标准管理**
- 自定义评价标准和限值
- 支持多种判定规则（≤、≥、区间）
- 🆕 **5 个国家标准预设模板一键导入**
  - GB 15618-2018 农用地土壤
  - GB 36600-2018 建设用地土壤
  - GB 3838-2002 地表水
  - GB/T 14848-2017 地下水
  - GB 5084-2021 灌溉水

✅ **数据评价**
- 根据选定标准自动评价
- 单指标和多指标综合评价
- 🆕 **土壤智能评价**（根据用地类型自动匹配限值）
  - 农用地：水田/果园/其他 + pH 分段
  - 建设用地：第一类/第二类
- 🆕 **筛选值和管制值两级判断**（GB 15618-2018）
  - 超过筛选值：中风险，需开展详细调查
  - 超过管制值：高风险，需采取风险管控或修复措施
- 🆕 **超标倍数计算**（所有指标统一支持）
- 实时显示评价结果

✅ **评价结果**
- 历史评价记录查询
- 详细评价结果展示
- 导出 Excel 报告

1. **配置评价标准** - 在「评价标准」模块添加评价指标和限值

| 备注 | ❌ | 其他说明信息 |

## 特色功能详解

### 🌾 土壤智能评价系统

系统完全支持 **GB 15618-2018** 和 **GB 36600-2018** 标准，可根据用地类型自动匹配污染物限值：

#### 农用地（GB 15618-2018）
- **用地类型**：水田 / 果园 / 其他
- **pH 分段**：<5.5, 5.5-6.5, 6.5-7.5, >7.5
- **两级限值管理**：
  - **筛选值**：超过此值表明可能存在风险，需开展详细调查（中风险）
  - **管制值**：超过此值表明风险较高，需采取风险管控或修复措施（高风险）
- **示例**：水田，pH<5.5，镉的筛选值为 0.3 mg/kg，管制值为 1.5 mg/kg

#### 建设用地（GB 36600-2018）
- **第一类用地**：居住用地、公共管理与公共服务用地等
- **第二类用地**：工业用地、商业服务业设施用地等
- **风险筛选值**：采用 GB 36600-2018 规定的风险筛选值
- **风险管制值**：按筛选值的 2 倍计算（参考方法）
- **示例**：建设用地第一类，砷的筛选值为 60 mg/kg，管制值为 120 mg/kg

### 📊 专用 Excel 模板生成

系统为每个国家标准自动生成专用的 Excel 导入模板：

| 模板类型 | 依据标准 | 指标数量 | 特殊字段 |
|----------|----------|----------|----------|
| 地表水模板 | GB 3838-2002 | 24 项 | - |
| 地下水模板 | GB/T 14848-2017 | 36 项 | - |
| 土壤模板 | GB 15618-2018 + GB 36600-2018 | 16 项 | 用地类型、农用地细分、pH 分段 |
| 灌溉水模板 | GB 5084-2021 | 22 项 | - |

### 🧪 真实数据验证

系统提供符合实际监测场景的测试数据集：
- **地表水**：10 个河流断面样品，22 项指标
- **土壤**：6 个地块样品（农用地 + 建设用地），15 项污染物
